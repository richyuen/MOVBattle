# AncientSandboxLike Map Spec

Use this map style for the single playable arena:

- Warm sand base terrain with subtle dunes
- Broken stone ruins and column fragments
- One main center engagement space
- Mild elevated flanks (left/right) for ranged advantage
- Team A and Team B placement zones on opposite ends
- Traversable routes around center obstacles to avoid path bottlenecks

Recommended dimensions:

- Total play bounds: `160 x 160` meters
- Team placement zone footprint each side: `50 x 40` meters
- Center open combat band: `40 x 40` meters

Keep colliders simple and NavMesh-friendly for mobile performance.
